Create game with visuals using QTrobot Studio

Overview

signal_cellular_alt  Level:  Intermediate
track_changes  Goal:  learn how to create a game with visuals using QTrobot Studio
task_alt  Requirements:

  •   Quick start with graphical programming

This video tutorial guides you towards creating a game with visuals (images) using QTrobot studio and run it on your QTrobot. You will learn how to upload your images on QTrobot Studio and use them in the game. You will learn how to use some of the basic blocks. We will use Show Says Act Block to make QTrobot speak and show facial expressions. We will be using also some of the Kid Tablet blocks. With Unclickable Images Block we will show images that learner can't click on them. To interact with learner we will use Clickable Images Block and respond accordingly. We will also use Reference Image Block. You can check the blocks below and continue with video tutorial.
