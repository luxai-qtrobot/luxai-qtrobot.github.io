Create Q&A game using QTrobot Studio

Overview

signal_cellular_alt  Level:  Basic
track_changes  Goal:  learn how to create a Q&A game with QTrobot Studio
task_alt  Requirements:

  •   Quick start with graphical programming

This video tutorial guides you towards creating a Q&A game using QTrobot studio and run it on your QTrobot. You will learn how to use some of the basic blocks. We have already used Start Block, which needs to be at the beginning of every game. We will use Show Says Act Block to make QTrobot speak and show facial expressions. You will learn how to use Select BLock to dictate the flow of the program and you will learn how to pause the execution of the program with Pause Block. You can check the blocks below and continue with video tutorial.
